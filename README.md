# DRC to Postie Chromium Extension


### **Important Notes**
- JSON file for category and type selection in extension is hosted on dfwinteractive.com which is owned and operated by the Dallas Regional Chamber.
- Emails sent to corresponding wordpress sites that have Postie installed also have the drc-postie-image-handler plugin installed that will download verified email attachments to the site itself if an image URL is sent.
- Any staff that uses this extension must have a google account *and* have their email whitelisted on postie for each of the listed sites.
